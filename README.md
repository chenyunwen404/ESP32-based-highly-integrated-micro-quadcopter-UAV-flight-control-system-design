# ESP32-S3 微型航控系统 (Micro Flight Control System)

本项目是一个高度集成的微型无人机飞行控制系统，旨在利用 ESP32-S3 的双核性能实现高频实时控制与无线通信的解耦。系统支持 8kHz 的超高采样与控制频率，集成了多种高精度传感器，并采用了级联 PID 控制算法。

## 1. 核心特性

- **双核架构并行处理**：
  - **Core 1 (高优先级)**：运行 `MotorControlTask`，执行 8kHz 的传感器读取、姿态解算和 PID 控制，确保飞行的极致实时性。
  - **Core 0 (低优先级)**：运行 `WIFIControlTask`，处理 AsyncUDP 通信、地面站遥测数据发送及电池电压监测。
- **高性能传感器驱动**：
  - **IMU (MPU6000/ICM42688)**：通过 SPI 总线连接，支持原始数据滤波（PT1 一阶低通滤波），极大降低了物理延迟。
  - **定高系统**：集成 BMP581 高精度气压计和 VL53L0X 激光测距仪，实现厘米级的室内外高度融合。
- **先进控制算法**：
  - **级联 PID**：采用“角度环+角速度环”的经典级联结构。
  - **D 项低通滤波**：针对微分项（D）引入 LPF，抑制电机高频震动带来的噪声。
  - **倾斜补偿 (Tilt Compensation)**：根据机身姿态自动修正油门输出，保持悬停高度。
- **动力管理**：
  - 支持 12 位分辨率（4096 级）的 PWM 电机驱动。
  - 支持 PWM 相位交错输出（90° 移相），降低系统电源瞬时压力。

## 4. 核心工作流程

1. **系统初始化**：`main.cpp` 中依次启动 WiFi AP、I2C 总线、校准 IMU 零偏，并启动双核任务。
2. **姿态解算**：利用 `IMUSolver` 结合加速度计和陀螺仪数据，通过互补滤波输出稳定的 Roll/Pitch/Yaw 角度。
3. **控制循环**：
   - 读取 `globalCmd`（来自 UDP 的遥控指令）。
   - 执行 `PIDMixContorl`，计算各轴 PID 输出。
   - 通过 `mixTable` 将控制量映射至四个电机的 PWM 占空比。
4. **安全机制**：
   - 油门检测：当油门低于阈值时自动重置 PID 积分项，防止地效累积。
   - 模式跳变检测：支持 WAIT, TAKEOFF, ACRO, STOP 四种飞行状态切换。
